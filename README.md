# VC Intelligence

A VC-style intelligence platform to discover and enrich startup data.

## Features
- Startup listing with details
- Company profile pages
- AI-based enrichment (summary & insights)
- Clean UI with Tailwind CSS

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- OpenAI API (optional)

## Getting Started
```bash
npm install
npm run dev
